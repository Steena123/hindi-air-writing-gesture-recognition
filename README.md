# Hindi Air-Writing Gesture Recognition using Leap Motion and PointNet++

A deep learning-based system for recognizing Hindi air-written characters using PointNet++ and Leap Motion Controller. This project processes 3D point cloud data captured from hand movements to classify Hindi characters without requiring physical contact.

## Overview

This project explores touchless handwriting recognition using hand motion captured by the Leap Motion Controller. The collected 3D point cloud data is processed and classified using the PointNet++ deep learning architecture.

## Features

- Touchless Hindi character recognition
- 3D hand tracking using Leap Motion
- Point cloud preprocessing
- PointNet++ based classification
- Deep learning pipeline
